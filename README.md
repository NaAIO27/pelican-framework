# Pelican Framework

Pelican 是一个高性能数据处理框架，专注于网络数据流的捕获、处理和分析。它提供了灵活的组件系统，允许用户构建复杂的数据处理流水线。

## 特性

- **模块化设计**：提供可扩展的组件系统，包括数据源、处理器和接收器。
- **高性能**：利用 Rust 的性能优势和异步编程模型，实现高效的数据处理。
- **安全性**：支持数据加密和完整性校验，确保数据在处理过程中的安全性。
- **监控与统计**：内置指标收集器，可以实时监控数据处理的性能和状态。
- **流量控制**：提供速率限制和背压控制机制，防止系统过载。

## 组件

- **数据源（Sources）**：支持内存数据源、重复数据源以及网络数据包捕获源。
- **处理器（Processors）**：包括过滤处理器、大小写转换处理器、数据包统计处理器、IP分析处理器和威胁检测处理器。
- **接收器（Sinks）**：提供控制台输出、统计信息收集和安全文件存储功能。
- **配置管理**：通过配置结构体管理流水线的配置信息。
- **安全模块**：提供数据加密、解密和完整性校验功能。
- **流量控制**：支持速率限制和背压控制，确保系统的稳定性和可靠性。

## 示例

项目包含多个示例程序，演示了如何使用 Pelican 框架构建数据处理流水线：

- `basic.rs`：基础示例，展示如何创建一个简单的数据处理流水线。
- `benchmark.rs`：性能基准测试示例。
- `config_example.rs`：配置管理示例。
- `minimal.rs`：最小化示例，展示最简配置下的数据处理流水线。
- `network.rs`：网络数据包捕获和处理示例。
- `performance.rs`：性能优化示例。
- `secure_pipeline.rs`：安全数据处理流水线示例。

## 使用方法

1. 安装 Rust 工具链。
2. 克隆仓库到本地。
3. 使用 Cargo 构建项目。
4. 运行示例程序以了解框架的使用方式。
